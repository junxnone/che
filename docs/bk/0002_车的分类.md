# 车的分类

  - 较为主流的分类方法包括根据动力分，根据空间分
  - **三厢**: 发动机舱/乘客舱/行李舱
  - **两厢车**: 乘客舱和行李舱打通

## 根据动力源分类

  - 油车
  - 电车
  - 混动
      - **插电混动**: 发动机可以参与动力输出
      - **油电混动**:
      - **增程式**: 没电了，发动机负责充电
