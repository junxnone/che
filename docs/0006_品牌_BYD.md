# 比亚迪
- 车系 - 王朝/海洋
- 汽油/纯电/混动
- 轿车/SUV
- 大小 - A00/A0/A/B/C/D
- 在售车
- 后缀 - `Pro/UP/Plus` - `入门/进阶/旗舰`
- 方向 - `DM-p/DM-i/EV` 
  - `Dual Mode` 双模式(插电混动) 
  - `DM-i` - `intelligent` 智能双模
  - `DM-p` - `powerful` 
  - `EV` - 纯电
